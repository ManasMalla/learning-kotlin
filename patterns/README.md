# Patterns

A series of programs to help break down the logic behind the pattern programs to help sharpen looping concepts (for loop).

Eg:

```
*****
***
*
***
*****
```

## Learning Objectives

- Sharpen looping concepts
- Develop critical thinking and problem solving
